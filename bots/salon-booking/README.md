# Telegram-бот записи в салон красоты

Демо для портфолио Kwork. Бот принимает онлайн-записи: выбор услуги → время → имя → телефон.

## Возможности

- Главное меню с услугами, контактами, инструкцией
- Пошаговая запись на услугу
- Сохранение заявок в `bookings.json`
- Команда `/bookings` — просмотр всех записей (для владельца)

## Запуск

1. Создай бота у [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен
2. Скопируй `.env.example` в `.env` и вставь токен:
   ```
   copy .env.example .env
   ```
3. Установи зависимости и запусти:
   ```
   pip install -r requirements.txt
   python bot.py
   ```

## Для заказчика

Все тексты (название салона, услуги, цены, адрес) легко меняются в начале файла `bot.py` — в словарях `SERVICES` и `TIME_SLOTS`.
