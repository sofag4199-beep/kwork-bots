# Telegram FAQ-бот для бизнеса

Демо для портфолио Kwork. Бот отвечает на частые вопросы клиентов — режим работы, меню, доставка, бронь.

## Возможности

- Меню с 6 типовыми вопросами
- Кнопка «Связаться с нами»
- Кнопка «Забронировать столик»
- Все тексты легко редактируются

## Запуск

1. Создай бота у [@BotFather](https://t.me/BotFather)
2. Скопируй `.env.example` в `.env`, вставь токен
3. Запусти:
   ```
   pip install -r requirements.txt
   python bot.py
   ```

## Для заказчика

Все вопросы и ответы — в словаре `FAQ` в начале `bot.py`. Меняешь тексты под любой бизнес: кафе, магазин, салон, клинику.
