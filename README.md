# VK-бот записи в салон

Демо-бот для группы ВКонтакте — запись на услуги с кнопками.

## Настройка группы ВК

1. Создай сообщество
2. **Управление → Сообщения** — включить сообщения
3. **Управление → Работа с API → Long Poll API** — включить
4. Тип события: **Входящее сообщение** + **Действие с кнопкой**
5. **Ключи доступа** — права: управление сообществом + сообщения

## Запуск

1. Заполни `.env` (см. `.env.example`)
2. Дважды кликни `start-bot.bat`

## Команды

- Клиенту: кнопки в чате или «начать»
- Владельцу: напиши боту **записи** — список заявок

## Файлы

- `bookings.json` — все заявки
- Уведомления владельцу приходят в личку ВК (OWNER_ID)
